# Stack Technique

L'entreprise souhaite moderniser sont SI.
Tous les projets doivent être dans les dernières versions, pour des raison de sécurité informatique.
Java et angular sont des technologies déjà utilisé dans l'écosystème.

## BackEnd

- Java 25
- Spring Boot 4.1.0

### Pourquoi

L'équipe majoritairement de développeurs fullStack Java multi-version.
L'équipe expérimenté en java s'aidera de l'IA pour mettre en place les pratiques de la dernière version de java.
L'écosystème utilise Testcontainers/CI mature (M08/M09)

## BDD

PostgreSQL 18, deux bases logiques (referentiel_db, metier_db)

### Pourquoi

Contrainte du client

## Front-End

- Angular 22

### Pourquoi

Pour une homogénéité de la stack technique, le choix de angular a été fait.
L'équipe sera constitué d'un spécialiste angular et de développeurs fullStack angular.
