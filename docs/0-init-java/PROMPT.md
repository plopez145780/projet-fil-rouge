# Prompt d'initialisation du projet en java 25

Rôle :

Tu es un développeur Java 25, spécialisé Spring Boot 4.0.1, chargé d'initialiser un projet backend simple et fonctionnel.

Contexte :

- Java 25 doit être utilisé comme version cible du projet.
- Spring Boot 4.0.1 est la version attendue du framework.
- Le dossier backend du projet s'appelle backend.
- Le projet doit rester minimaliste et orienté "Hello World".
- Le but est d'obtenir une application qui compile et s'exécute correctement avec Java 25.

Contraintes :

- Respecter strictement la version Java 25.
- Utiliser un projet Maven compatible avec Java 25.
- Créer un point d'entrée Java simple dans le dossier main.
- Éviter toute complexité inutile : pas de contrôleur web, pas de dépendances superflues.
- Vérifier que le build passe avant de conclure.
- Ne pas committer de secrets ni de configurations sensibles.

Actions :

Créer une application Hello World en Java 25 dans le dossier backend, en respectant les versions demandées et en produisant un résultat compilable et exécutable.

Plan :

- Créer un point d'entrée Java dans le main avec un programme Hello World.
- Créer le fichier pom.xml adapté à Java 25 et Spring Boot 4.0.1.
- Ajouter la structure de base du projet Maven si nécessaire.
- Compiler le projet avec Maven.
- Exécuter l'application si possible pour valider le comportement.
- Vérifier que le build est réussi.
- Rendre un résumé clair des fichiers créés et du résultat final.
