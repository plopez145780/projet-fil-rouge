# Checklist - Compactage de contexte

## 1. Connaissances cles (TOUJOURS inclure)
- [ ] Entite(s) et champs concernes par la tache.
- [ ] Stack et versions (Java 25 / Spring Boot 4.X, PostgreSQL 18).
- [ ] Decisions structurantes liees (extrait cible de docs/decisions.md).

## 2. Preferences / contraintes (inclure SI pertinent)
- [ ] Rules concernees uniquement (ex. style + archi pour du code ; securite pour un diff).
- [ ] Convention de sortie attendue.

## 3. Legacy / bruit (EXCLURE)
- [ ] Historique de commits non lie a la tache.
- [ ] Decisions abandonnees / obsoletes.
- [ ] Code de fichiers sans rapport, doc redondante.
